## 1. 需求与背景

### 1.1 业务场景

资产管理系统需要提供折旧计算功能，帮助企业用户：
- 在资产详情页查看单笔资产的折旧报表
- 支持按**直线法（Straight-Line Method）** 计算折旧
- 支持按**双倍余额递减法（Double Declining Balance Method）** 计算折旧
- 后台定时任务自动更新日累计折旧数据，确保报表数据实时性

### 1.2 核心计算规则

| 方法 | 公式 | 说明 |
|------|------|------|
| **直线法** | `(原值 - 残值) / 使用年限` | 每年折旧额固定 |
| **双倍余额递减法** | `年折旧率 = 2 / 使用年限；年折旧额 = 期初账面价值 × 年折旧率` | 前期折旧多，后期转为直线法 |

### 1.3 依赖关系

| 模块 | 依赖类型 | 说明 |
|------|----------|------|
| `asset-core` | 强依赖 | 资产主数据实体（Asset Entity） |
| `asset-depreciation` | **本次开发** | 折旧计算核心模块 |
| `audit-service` | 弱依赖 | 折旧变更审计日志 |

## 3. 边界约束

### 3.1 作用域限定

```
┌─────────────────────────────────────────────────────────┐
│  ✅ 在范围内                                             │
├─────────────────────────────────────────────────────────┤
│  - 直线法折旧计算                                         │
│  - 双倍余额递减法折旧计算                                  │
│  - 折旧数据表 (depreciation_schedule)                     │
│  - 定时任务 (每晨 02:00 UTC 更新)                          │
│  - 资产详情页折旧报表 API                                 │
│  - 累计折旧与账面价值计算                                  │
│  - 部分月份首年折旧计算                                   │
└─────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────┐
│  ❌ 超出范围                                             │
├─────────────────────────────────────────────────────────┤
│  - 年数总和法（暂不支持）                                  │
│  - 税务合规校验逻辑                                       │
│  - 多币种换算                                            │
│  - 折旧凭证自动生成（划入 Phase 4）                        │
│  - 资产减值处理                                           │
│  - 历史期间重述                                           │
└─────────────────────────────────────────────────────────┘
```

### 3.2 技术约束

| 约束项 | 规格 | 验证方式 |
|--------|------|----------|
| **计算精度** | 结果保留 **2 位小数**，采用 **ROUND_HALF_UP** | ATB-1-01 |
| **并发控制** | 定时任务采用分布式锁，防重复执行 | ATB-4-02 |
| **批量上限** | 单次批量处理上限 **10,000 条**资产记录 | ATB-3-01 |
| **超时阈值** | 单笔资产折旧计算超时 **500ms** | 性能测试 |
| **状态过滤** | 仅对 `status='ACTIVE'` 且 `acquisition_date` 不为空的资产执行折旧 | ATB-3-03 |

### 3.3 数据约束

| 约束项 | 规则 |
|--------|------|
| **原值** | 必须 > 0 |
| **残值** | 必须 ≥ 0 且 < 原值 |
| **使用年限** | 必须 > 0，支持小数（按月计算） |
| **购置日期** | 不能晚于当前日期 |

## 6. 附录

### 6.1 关键数据结构

#### DepreciationRecord Entity

```python
class DepreciationRecord:
    """折旧记录实体"""
    id: int
    asset_id: int                          # 关联资产ID
    period_year: int                       # 折旧期间-年
    period_month: int                      # 折旧期间-月
    method: str                            # "STRAIGHT_LINE" | "DOUBLE_DECLINING"
    depreciation_amount: Decimal          # 当期折旧额
    accumulated_depreciation: Decimal      # 累计折旧
    book_value: Decimal                    # 账面价值
    calculated_at: datetime                # 计算时间
```
