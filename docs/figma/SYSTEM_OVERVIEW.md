# 企业资产管理系统 - 交互原型

## 系统简介

这是一个功能完整的企业资产管理系统交互原型,基于您提供的需求文档开发,涵盖资产全生命周期管理的核心功能。

## 技术栈

- **前端框架**: React 18.3.1
- **路由管理**: React Router 7 (数据模式)
- **样式方案**: Tailwind CSS v4
- **图表库**: Recharts
- **图标库**: Lucide React
- **构建工具**: Vite

## 使用说明

### 导航结构
- `/` - 仪表板
- `/assets` - 资产台账
- `/equipment` - 重要设备
- `/inventory` - RFID盘点
- `/idle` - 闲置资产
- `/compensation` - 资产赔偿
- `/approval` - 审批流程
- `/analytics` - 数据分析
- `/settings` - 系统设置

### 主要交互流程

1. **新增资产流程**
   - 进入资产台账页面
   - 点击"新增资产"按钮
   - 填写资产信息
   - 保存并同步ERP

2. **RFID盘点流程**
   - 创建盘点任务
   - 开始RFID扫描
   - 实时查看盘点进度
   - 处理账实差异
   - 完成盘点并生成报告

3. **设备保养流程**
   - 查看保养提醒
   - 创建保养记录
   - 记录保养详情
   - 自动更新设备状态

4. **审批流程**
   - 查看待审批事项
   - 审查申请详情
   - 批准/驳回操作
   - 查看审批历史
