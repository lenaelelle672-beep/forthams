保留 React/Vite 的原因

当前项目已有 React Router、Tailwind v4、Radix/shadcn、测试资产和页面体系。切 Nuxt 会重写大量边界，视觉收益并不自动增加。

借鉴 Nuxt Frontend Design Pro 的部分

  • 统一 API client、错误处理、鉴权失效跳转。
  • 表格模式：筛选、列配置、批量操作、导入导出、空/加载/错误状态。
  • DTO 对齐、权限 Hook、i18n、暗黑模式、大屏独立路由。

可选试点
